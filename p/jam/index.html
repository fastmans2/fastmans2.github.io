FastGame Jam

The Edge

Theme:

Choice Based Game

  • You can use any programming language
  • Deadline is December 1 2019 22:00 CET
  • You must follow the rules but guidelines can be ignored if it helps the game
  • Groups are allowed

General

Rules

  • Imagery must be PG (12+ but they can be some pretty tough twelve year olds)
  • Extremely rapid changes in brightness or contrast must have an epilepsy warning
  • Horror (mainly jump scares) is allowed but it must be indicated at the start of the game

Guidelines

  • The controls should be only buttons/clickable text
  • The game should have multiple endings
  • You should make as many aspects of the game(visuals, audio, code) by yourself but you can use anything if it doesn't break any rules

Programming Environment Rules

Web Based

  • You must provide the whole folder of your project and a starting page called "index.html"

Program Based

  • You must provide the .exe of your game that starts the game and doesn't require an install

Getting started guide

The most basic kind of game is just a bunch of pages leading to one another.
This game jam was designed around that concept and it should be pretty easy to make so you can focus on the story elements of your game
